BANK ACCOUNT SYSTEM
This is a simple python project that simulates a bank account system.
## features
  deposit money into an account
  withdraw money from your account
  check your account balance
  PIN verification
## How to run 
  Make sure you have python installed (Greatly recommend Python 3)
  Download the Bank-syetem.py file.
  open a terminal or command prompt in folder where Bank-system.py is located
  Run the program:
......bash
python bank-system.py
